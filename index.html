Intentzio onenarekin, jakina

Ikastaro honetan zerotik abiatuko gara, ez da aldez aurretik programazioaz ezer jakin behar, ikasteko gogoa besterik ez da behar. Hauexek dira ikastaroaren lerro nagusiak:

  • HTML5, CSS3 eta JavaScript irakastea, haien xehetasunak aztertuz, ikasiz eta erabiliz.
  • Zalantzak argitzeko tokiak eta helbideak non diren erakustea.
  • Mundu errealeko profesionalek nola lan egiten duten erakustea, haien tresnak aurkeztuz, hala nola Github, Kode editoreak, nabigatzaileen xehetasunak edota zerbitzari lokalak.

Programazioaren munduan arazo bera konpontzeko modu asko daude, guztiak balekoak. Halere, ikastaroaren helburua ez da problema bat ebazteko modu guzti-guztiak ikastea (hori urteetako praktikarekin ikasten da)..

Ikasmateriala osatzen duten ikasgaiak

1.- Web garapena

Internet zer den ikusita, web-orriak sortzeko eta webguneak osatzeko beharko ditugun elementuak eta lanabesak ikusiko ditugu hiru ataletan: WEBGUNEAK, LENGOAIAK eta LANABESAK.

3.- HTML

Web-orrien edukia eta egitura, irudiak, zerrendak, bideoak eta abar definitzeko erabiltzen den HTML lengoaiaren xehetasunak ikasiko ditugu hiru ataletan: EGITURA, EDUKIA eta FORMULARIOAK.

4.- CSS

Web-orria pantailari egokitzeko 'Media Queries' izeneko tresnaz baliatuko gara. Tresna hori CSSk hornitzen du.

5.- CSS orrien diseinua

Web-orria pantailari egokitzeko 'Media Queries' izeneko tresnaz baliatuko gara. Tresna hori CSSk hornitzen du.

6.- CSS diseinu moldagarria

Gure web-orriak dispositibo guztietan modu erakargarrian erakusteko erabili behar dugun kodea.

7.- JavaScript

JavaScriptek jarraibideak modu sekuentzialean exekutatzen ditu, sistemari adierazteko zer nahi dugun egin.

8.- AJAX

'XMLHttpRequest' objektua da JavaScript-etik informazioa lortzeko, orri osoa berriro kargatu beharrik gabe.

9.- API (1)

  • Form: formularioak kudeatzeko.
  • Stream: Aparatuarekin konektatzeko.
  • Fullscream: pantaila osoa.
  • Canvas: mihisea erabilktzeko.

10.- API (2)

  • Pointer Lock: xaguaren funtzioak.
  • Web Storage: datuak gordetzeko.
  • IndesedDB: datu ugari gordetzeko.
  • File: norberaren fitxategiak erabiltzeko.

11.- API (3)

  • Drag and Drop: elementuak mugitzeko.
  • Geolocation: kokapenaz jolasteko.
  • History : bilaketak gordetzeko.
  • Page Visibility: gastua kudeatzeko.

12.- API (4)

  • Web Messaging: mezuak kudeatzeko.
  • Web Socket: komunikazioa bideratzeko.
  • Web RT: bapateko komunikazioak.
  • Web Audio: audioa kudeatzeko.
  • Web Workers: albo prozesuak.
